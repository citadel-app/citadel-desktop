## 🚀 Quick Start

### Open in Citadel

1. **Download** [Citadel](https://github.com/citadel-app/citadel-desktop/releases/latest) (portable executable, no installation needed)
2. **Launch** the app and sign in with GitHub
3. **Choose** "Clone Repository Here" or paste the deep-link in your browser: `citadel://clone?url={{CLONE_URL}}`

### Manual Clone

```bash
git clone {{CLONE_URL}}
```

Then open the cloned folder within the Citadel application.

## 📖 What is Citadel?

Citadel is a local-first, Git-backed knowledge workspace with:

- 📝 **Rich document editing** — Markdown, LaTeX, Code notebooks, and Canvas Whiteboards
- 🧠 **AI-powered** search, coding assistance, and content generation
- 🔄 **Git sync** — your data lives in a standard `.git` repo you own and control
- 🏗️ **Hackable** — fully configurable entry types and UI modules
- 🔒 **Private** — everything runs locally on your machine, with no required cloud dependency

## 📁 Workspace Structure

This repository contains a `.codex` configuration directory that defines the workspace structure. Citadel reads this repository to organize your entries, metadata schemas, and custom views.

The `README.md` and associated visual assets (such as banner images) are designed to be standard GitHub-flavored Markdown.
